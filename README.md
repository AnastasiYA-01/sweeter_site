# 🍰 Sweeter — Веб-приложение для кондитеров

> Сладкий инструмент для приёма заказов и продвижения кондитерской продукции

## 🧁 О проекте

**Sweeter** — это лёгкое и стильное веб-приложение, созданное для частных кондитеров.  
Оно помогает принимать заказы онлайн, показывать каталог десертов и общаться с клиентами в удобном формате.

Проект разработан студентами 2 курса в рамках учебной дисциплины «Основы проектной деятельности».

## 🎯 Цель

Создать бесплатный и простой инструмент, который заменит хаос в мессенджерах и поможет кондитерам:

- 📸 красиво презентовать свои изделия  
- 📦 принимать заказы в 3 клика  
- 💬 общаться с клиентами через встроенный чат  

## 🖼️ Что внутри

| Экран | Описание |
| :--- | :--- |
| 🏠 Главная | Баннер, категории, популярные товары |
| 📚 Каталог | Сетка карточек с фото, ценой и кнопкой «В корзину» |
| 🍰 Карточка товара | Крупное фото, описание, состав, цена |
| 📝 Форма заказа | Поля для имени, телефона, адреса, комментария |
| 💬 Чат | Диалог с кондитером и быстрые вопросы |

## 🛠️ Технологии

- HTML5  
- CSS3  
- JavaScript (ES6)  
- GitHub Pages — для публикации
