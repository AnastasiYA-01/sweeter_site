# 🍰 Sweeter — Веб-приложение для кондитеров

> Сладкий инструмент для приёма заказов и продвижения кондитерской продукции

---

## 🧁 О проекте

**Sweeter** — это лёгкое и стильное веб-приложение для частных кондитеров.  
Оно помогает принимать заказы онлайн, показывать каталог десертов и общаться с клиентами.

Проект разработан студентами 2 курса в рамках дисциплины «Основы проектной деятельности».

---

## 🎯 Цель

Создать бесплатный и простой инструмент, который заменит хаос в мессенджерах:

- 📸 красиво презентовать изделия
- 📦 принимать заказы в несколько кликов
- 💬 общаться через встроенный чат
- 🔐 авторизация пользователей
- 👩‍🍳 админ-панель для управления заказами и отзывами

---

## 🖼️ Экраны

| Экран | Описание |
| :--- | :--- |
| 🏠 Главная | Баннер, категории с выпадающими списками, популярные товары, отзывы |
| 📚 Каталог | Сетка карточек с фото, ценой и кнопкой выбора количества |
| 🍰 Карточка товара | Крупное фото, описание, состав, цена |
| 📝 Оформление заказа | Форма с адресом и телефоном, отображение корзины |
| 💬 Чат | Диалог с кондитером и быстрые вопросы |
| 👤 Профиль | Личные данные и история заказов |
| 🔐 Авторизация | Регистрация и вход с сохранением данных |
| 📋 Админ-панель | Управление заказами и модерация отзывов (пароль: `простой`) |

---

## 🛠️ Технологии

- **Frontend:** HTML5, CSS3, JavaScript (ES6)
- **Backend:** Node.js, Express
- **База данных:** PostgreSQL (Docker)
- **Хранение:** localStorage (для корзины), API (для пользователей)
- **Деплой:** GitHub Pages (статическая часть)
